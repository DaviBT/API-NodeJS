# API com NODEJS

## Função
API que permite o cadastro, login e exclusão do usuário assim como criação, edição e exclusão de produtos ou pedidos.
A API não foi publicada devido ao preço do Microsoft AZURE.

### Tecnologias usadas
* Node, NPM e VS Code
* Nodemon
* CRUD REST
* Rotas
* Controllers
* MongoDb
* Mongooose
* Async/Await
* Autenticação
* Autorização
* CORS
